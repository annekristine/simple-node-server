The First Internet

“This ARPA program has created no less than a revolution in computer technology and has been one of the most successful projects ever undertaken by ARPA. The full impact of the technical changes set in motion by this project may not be understood for many years.”
-- from ARPANET Completion Report, January 3, 1978

ARPANET became the foundation of the modern internet; internet as a word wasn’t even known before. It was developed in 1969 that’s why the generation today might not even know what it is. ARPA (Advanced Research Project Agency) has a goal of linking computers together to decentralize the information storage and thus, brilliant people were able to create the first internet.

It uses the packet switching system for sending files wherein these are broken into small pieces called packets and travel using different routes to reach their destination. When this method is compared with circuit switching, it is more time consuming, but is good for security purposes. Since it is not in order when received, it takes longer time to assemble but it lessens the success or probability of a hacker to receive the whole file.


TIMELINE

To better understand how ARPANET was created, here’s a timeline of events.

1962

- J.C.R. Licklider’s concept called Intergalactic Network where he envisioned an interconnected network where everyone can access files or communicate with each other anywhere.

1963

- JCR Licklider contracted organizations like UCLA, MIT and BBN to give life to his concept, to create the “internet”.
- ASCII (American Standard Code for Information Exchange) was created to regulate the machines from different manufacturers for exchanging data. It was the first universal standard for computers.

1964

- Development of Multics timesharing operating system.

1965

- PDP-8, the first commercially successful minicomputer that was small enough to be placed on a desktop, was unveiled
- The first wide area connection which was funded by ARPA was created by Larry Roberts and Thomas Marill.
- JOSS(Johnniac Open Shop System) goes online which allow online computational problem solving.

1966

- Bob Taylor with the help of Charles Herzfeld, was able to connect 3 computer systems research sites around the nation by linking all the contractors.

1967

- There was a conference that brought all the ARPA researchers together. One suggested using the Interface Message Processors (IMP) to manage the interconnected network. This processor evolved into the routers that we have today.

1968

- The network measurement system at the UCLA was prepared by Kleinrock’s system. This became the site of the first node.

1969

- In October 30, the first message was sent over the internet. Kleinrock supervised Kline, a student in the setting up of a message transmission from UCLA host to Bill Duvall’s host. The message “login” was the intended one but only “lo” was received since the system crashed on “g”.

1970

- Nodes were added to the ARPANET one per month.
- NCP(Network Control Progam) was created, the initial ARPANET host-to-host protocol.

1971

- The Telnet protocol was completed while the FTP was still in the works.
- Intel released the 4004, the first computer on a chip.

1972

- Ray Tomlinson began developing the program of enabling e-mail be sent over the ARPAnet. He also developed the user@host convention.

1973

- 30 institutions were connected with ARPAnet
- Bob Kahn and Vint Cerf proposed the TCP (Transmission Control Protocol).
- Bob Metcalfe was working on a wire-based system which eventually became Ethernet.

1976

- Seymour Cray demonstrated the first-vector processor supercomputer called CRAY-1.

1978

- The ARPANET experiment was formally completed.
