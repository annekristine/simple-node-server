Early World Wide Web


World Wide Web (WWW) is an information space where web resources, files and other documents are identified by URLs, which can accessed through the internet. It was also known as the Web. According to the TNW News, “World Wide Web was adopted as the project’s name following rejected possibilities such as ‘The Mine of Information’ and ‘The Information Mesh’”. The man who created this pillar of the internet that we have today is Tim Berners-Lee. WWW was invented in 1989 by this British scientist in CERN. The goal of creating this web was originally for research communication purposes, for meeting the demand for automatic-information sharing for the scientists in the world.

The first web browser software in the world was hosted at Lee’s NeXT computer in 1990. His first web page can still be accessed today but of a later version already. You can check the later version by clicking this. His website described the basic features of the web which includes how to access the other’s documents and how to create a server.

Source: http://www.w3c.org.ma/Talks/Med-It-2014/Med-It-2014.html

In developing the World Wide Web, he was also able to develop important technologies like the URL (Uniform Resource Locator), UDI (Universal Document Identifier) and URI (Uniform Resource Identifier), HTML (Hypertext Mark-up Language) and HTTP (Hypertext Transfer Protocol).

Source: http://geospatial.blogs.com/geospatial/2011/08/aug-6-1991-the-worlds-first-web-site.html

For people to easily access it, CERN announced in 1993 that WWW will be free for everyone, no fees needed to pay.
